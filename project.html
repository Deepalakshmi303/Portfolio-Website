AI-Based Classification and Detection of Higher Stress Levels

Technologies: Python, Flask, Machine Learning, MySQL

Description: Developed an AI-powered stress detection system capable of analyzing student behavior patterns and predicting elevated stress levels. The project integrates Python-based machine learning models with a Flask web application to process student input data, evaluate emotional indicators, and classify stress categories. It includes modules for administrators, students, and faculty for monitoring stress trends, viewing risk scores, and generating analytical insights. The emotion-detection component assists teachers in identifying early warning signs, enabling timely intervention and improved mental-health support.

Smart Parking System

Technologies: Java, JDBC, MySQL

Description: Designed and developed a Smart Parking Management System that enables users to track and reserve available parking spaces in real time. The backend, built using Java and JDBC, manages core functionalities including slot allocation, user authentication, entry/exit monitoring, and automated billing. A MySQL database handles parking-slot metadata, reservation details, and administrative configurations. The system also features an admin dashboard for managing users, updating pricing, and overseeing system performance, improving urban parking efficiency and reducing congestion.
